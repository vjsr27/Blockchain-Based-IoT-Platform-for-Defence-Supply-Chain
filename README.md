# Blockchain-Based IoT Platform for Defence Supply Chain

## Overview
This project presents a blockchain-based IoT platform designed to improve transparency, security, and traceability in defence supply chain operations. The system uses a private permissioned blockchain to enable secure transactions and decentralized collaboration among supply chain stakeholders.

## Key Features
- Secure and transparent transaction management
- Decentralized supply chain data sharing
- Improved traceability of defence components
- Smart contract-based validation

## Technologies Used
- Blockchain
- IoT
- Smart Contracts
- Distributed Systems

## Use Case
The model demonstrates how blockchain technology can enhance trust, automation, and efficiency in complex defence supply chain networks.
